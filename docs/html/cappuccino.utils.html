cappuccino.utils.util.confInt(x, percentile=68.27, onesided=False, prior=None)¶

Calculates the confidence interval for a posterior sample x.

Parameters:
  • x (numpy array) – 1D array of the posterior samples
  • percentile (float) – Percentile for which to calculate confidence interval. Defaults to 68.27.
  • onesided (float) – Standard confidence interval (False, None), upper limit (‘lower’, ‘hi’), or lower limit (‘upper’, ‘lo’)
  • prior (list, length 2) – Prior range for the parameter, recommended if ‘onesided’ is not False.
Returns:

Confidence interval in a numpy array, formatted as [median, lower limit, upper limit]
